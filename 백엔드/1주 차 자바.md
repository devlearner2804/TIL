# <h1> 자바란?
자바는 컴퓨터에게 명령을 내리는 프로그래밍 언어이다. 특징은 객체지향 언어이고, 한 번 작성하면 여러 환경에서 실행할 수 있으며, 앱이나 웹, 프로그램을 만드는 데 많이 사용된다는 점이다, 이를 객체지향 프로그래밍 언어 라고 한다.

# <h3> 자바의 자료형 메모리 공간 범위
정수형

byte: -127 - 127 (1byte, \(2^{8}\))

short: -32,768 - 32,767 (2byte, \(2^{16}\))

int: -2,147,483,648 - 2,147,483,647 (약 20억) (4byte, \(2^{32}\))

long: -9,223,372,036,854,775,808 - 9,223,372,036,854,775,807 (8byte, \(2^{64}\))

실수형

float: 대략 -3.4E38 - 3.4E38, 7자리 정밀도 (4byte, \(2^{32}\))

double: 대략 -1.7E308 ~ 1.7E308, 15자리 정밀도 (8byte, \(2^{64}\))

기타

boolean: true, false

# <h2> 주석
주석(comment) 
주석이란 프로그램을 실행에 영향이 ㅇ없는 설명 글 인데
자바 에서는 //주석처리 
이런 식으로  // 을 하면 한줄이 주석 처리가 되고 /* */ 을 한다면 여러줄이 주석 처리가 된다.

# <h2> if문

# <h3> if문 이란
  (조건) {
    실행할 코드
}
조건이 true면 실행, false면 무시
```
int age = 20;

if (age >= 19) {
    System.out.println("성인입니다");
}
```
# <h4> else if
조건을 추가하는데
앞 조건이 틀리면 새로운 조건을 검사한다.
```
int age = 17;

if (age >= 19) {
    System.out.println("성인");
} else if (age >= 13) {
    System.out.println("청소년");
}
```

# <h2> do while문
조건이 True인 동안 계속 반복을 하는데

do while 문은 조건과 상관없이 최소 1번은 무조건 실행이 된다.
```
do {
    System.out.println("현재 숫자는: " + i);
    i++;
} while (i < 3);
```

# <h2> 클래스
자바는 객체를 만들기 위해 반드시 클래스를 먼저 만들어야 한다. 클래스는 객체를 만들기 위한 일종의 ``틀``이다

public class Car{

}


# <h2> String클래스
``문자열을 표현``하는 자바에서 가장 많이 사용하는 클래스


String str1 = "hello";
String str2 = "hello";


# <h2> 필드(field)선언
자바에서 ``속성``을 ``필드(field)``라는 용어로 사용한다.

public class Car{
String name;
int number;
}


# <h2> 메소드란?
필드가 물체의 상태라면, 물체의 ``행동``에 해당하는게 메소드.





# <h2>패키지
패키지란 서로 관련이 있는 클래스 또는 인터페이스들을 ``묶어 놓은 묶음``이다.




# <h2>클래스 형변환
자식객체가 가지고 있는 메소드나 속성을 사용하고 싶다면 형변환 해야 한다.

# <h2>추상 클래스
추상 클래스란 구체적이지 않은 클래스를 의미한다.

# <h2>인터페이스
인터페이스: 서로 관계가 없는 물체들이 상호 작용을 하기 위해서 사용하는 장치나 시스템.
