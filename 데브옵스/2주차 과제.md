## 저장소(Repository)
프로젝트 시작 & github 연결

- git init

현재 폴더를 git 저장소로 생성 (.git 생성)

- git clone 주소

GitHub 저장소를 내 컴퓨터로 복사

- git remote -v

연결된 원격 저장소 주소 확인하는 명령어

## 스테이징(Staging)

    
 커밋 전에 파일 올리는 단계
    
   * git add 파일명
        
       → 특정 파일 추가
        
   * git add .
        
        전체 파일 추가
        
   * git reset 파일명
        
     add 취소   

## 커밋(Commit)

변경사항 저장 (버전 생성)

* git commit -m "메시지"
  
  변경사항 저장

* git commit -am "메시지"
  
  add + commit (추적된 파일만)


## 브랜치(Branch)

작업 분리 (협업 핵심)

* git branch
  
  브랜치 목록 확인

* git branch 이름
  
  브랜치 생성

* git checkout 이름
  
  브랜치 이동


## 병합(Merge)

브랜치 합치기

* git merge 브랜치명
  
  현재 브랜치에 다른 브랜치 병합

* 충돌(conflict)
  
  같은 파일 수정 시 발생, 직접 수정 필요


## 원격 관리(Remote)

GitHub 연결 상태 관리

* git remote show origin
  
  원격 저장소 상세 정보 확인

* git remote remove origin
  
  원격 연결 제거


## 임시 저장(Stash)

작업 잠깐 보관

* git stash
  
  현재 작업 임시 저장

* git stash pop
  
  다시 가져오기
